## 功能

- **🎬 智能配音** — 单人配音、多人对话、情感配音三种模式随心切换。349+ 角色一键搜索筛选，支持收藏、自定义配音员。情感标签从文件名自动解析，开心生气委屈害羞应有尽有
- **🎭 一句话克隆** — 传一段参考音频，AI 学舌比谁都快，说得比你还像你（需要后端小老弟 tts_serve_mlx 配合）
- **🎨 声音设计** — 不用调参数，写一句"温柔知性的成熟女声"，AI 就给你现场捏一个。支持高级参数（推理步数、CFG）
- **📦 自定义配音员** — 声音设计合成的音频一键加入配音员列表，像普通角色一样使用
- **🎛️ 声音工坊**（2026-07-13新增） — 上传任意音频，加实时 DSP 效果（回声/变声/失真/混响等 13 种效果），一键导出经效果链处理的 WAV。全部效果原生 Web Audio 映射，离线渲染稳定不静音
- **💬 聊天气泡多人配音**（2026-07-14新增） — 告别老式下拉框，聊天气泡编辑对话，点角色自动选情感，合成一气呵成
- **⚙️ 偏好设置** — 换主题、切语言、清缓存，怎么舒服怎么来
- **🛡️ 错误边界** — 页面崩了也不怕，一键满血复活，连作者自己都经常用到这个功能

```bash
NODE_ENV=development npm start     # Electron
```
