## 功能

- **🎬 智能配音** — 单人独白、多人嘴炮、情感爆发三种模式，想咋配咋配。349+ 角色库随便翻，还能收藏加自定义。情感标签自动从文件名里猜（开心生气委屈害羞），不用你手动写
- **🎭 一句话克隆** — 扔一段参考音频进去，AI 学舌比谁都快，说得比你还像你（需要后端小老弟 tts_serve_mlx 在后台搬砖）
- **🎨 声音设计** — 不用拧旋钮调参数，写一句"温柔知性的成熟女声"，AI 就给你现场捏一个。高级玩家还能调推理步数和 CFG
- **📦 自定义配音员** — 声音设计捏出来的音频一键入库，跟原生角色一样用，不用跪着求人
- **🎛️ 声音工坊**（2026-07-13新增） — 找段音频扔进去，加特效（回声变声失真混响…一共 13 种），一键导出带效果的 WAV。所有效果原生 Web Audio 实现，导出不会给你一坨静音
- **💬 聊天气泡多人配音**（2026-07-14新增） — 终于把那破下拉框编辑器毙了。像聊微信一样写台词，点角色自动带情感，合成一条龙
- **⚙️ 偏好设置** — 换皮肤切语言清缓存，怎么舒服怎么来
- **🛡️ 错误边界** — 页面崩了也不怕，一键满血复活。连作者自己都经常用到这个功能（真的）
