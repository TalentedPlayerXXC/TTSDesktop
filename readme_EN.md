# QwenTTS Desktop

A desktop dubbing client built with Electron + React, offering three core capabilities: smart dubbing, voice cloning, and voice design.

## Features

- **Smart Dubbing** — Select a voice actor (filterable by gender, style), enter text, and synthesize with one click
- **One-Shot Cloning** — Upload a reference audio clip to clone a target voice rapidly
- **Voice Design** — Craft a custom voice style using prompts plus adjustable parameters (speed, volume)
- **Preferences** — Personalize theme, language, notification settings, and more

## Tech Stack

Electron · React 19 · TypeScript · Vite 6 · Ant Design 6 · react-router 7 · Axios

## Getting Started

```bash
# Build and launch
npm run preview

# Or development mode (two terminals required)
npm run dev                        # Vite dev server
NODE_ENV=development npm start     # Electron
```
