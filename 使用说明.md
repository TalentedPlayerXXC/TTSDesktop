## 1. 安装

### macOS

1. 双击打开 `绘声-1.0.0-arm64.dmg`
2. 将 **绘声** 拖入 Applications 文件夹
3. 首次打开若提示「来自身份不明的开发者」→ **右键应用 → 打开**
4. 若仍提示文件损坏 → 双击 **修复.command**

### 注意事项

- 首次启动加载 ML 模型（约 5.2GB），稍等
- 确保 8000 端口未被占用：`lsof -ti:8000 | xargs kill`

---

## 2. 快速开始

### 2.1 启动应用

双击「绘声」图标。启动过程：

1. ✅ 前端加载
2. ✅ Python TTS 后端启动
3. ✅ 296+ 角色数据加载
4. ✅ 模型加载

### 2.2 选择角色

- **关键词搜索** — 输入角色名
- **声线筛选** — 全部 / 男声 / 女声
- **标签筛选** — 声线/气质标签，可展开收起
- **收藏** — 星标收藏

## 3. 智能配音

### 3.1 单人配音 🎤

1. 输入台词
2. 选择情感（可选）
3. 点击角色
4. 合成试听

### 3.2 多人配音 🗣️

1. 添加角色行 → 选人 → 输台词
2. 合成（自动合并音频）

### 3.3 情感配音 🎭

1. 输台词 → 选角色 → 选情感
2. 合成（使用 VoxCPM2 模型）

---

## 4. 一句话克隆 🎭

输入角色名 → 输目标台词 → 合成。

**提示：** 参考音频越清晰效果越好。

## 8. 常见问题

### Q: 安装后打不开？

**A:**
- 提示「来自身份不明的开发者」→ **右键 app → 打开**
- 提示「文件损坏」→ 双击 **修复.command**
- 端口占用 → `lsof -ti:8000 | xargs kill`

### Q: 合成没声音？

1. 检查模型是否已加载（看终端日志）
2. 检查系统音量
3. 换角色或情感试试

### Q: 模型加载失败？

- `tts_serve_mlx/models/` 是否完整（~5.2GB）
- 查看终端日志

### Q: 自定义配音员不见了？

设置中点「清空并重建」触发同步。

### Q: 语音质量不好？

- 克隆：用清晰无噪音的参考音频
- 声音设计：调整 steps/CFG
- 情感：确保角色有匹配的情感音频
